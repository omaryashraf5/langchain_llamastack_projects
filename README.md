# LangChain LlamaStack Projects

A collection of projects demonstrating AI-powered applications using LangChain and LlamaStack.

## Projects

### Executive Dashboard
An intelligent executive dashboard that provides natural language querying capabilities for sales data analysis. Built with LangChain, LlamaStack, and Streamlit.

**Features:**
- Natural language queries for sales data
- Interactive visualizations
- Multiple data sources integration
- AI-powered insights
